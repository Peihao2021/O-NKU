开启之后，sitemap.xml地图以及全站URL将带有https头协议

开启前，请先在空间正确配置SSL证书。
在能正常访问https://域名的情况下，才开启这个功能，使整站的URL都强制采用https协议访问。

 
 

追加之后，生成静态模式下，自动在PC端模板头部head标签出现一段PC跳转手机端的JS代码，当在手机端访问PC端URL时，自动跳到相应的手机端URL。

 

开启之后，将在发布/编辑文档的页面出现选择副栏目，该功能会过多消耗性能，请悉知。
