伪静态规则：

location / {
if (!-e $request_filename){
rewrite sitemap.xml /index.php?m=plugins&c=Pushall&a=sitemap last;
rewrite ^\/(\w*)\/([1-9]\d*)\.xml$ /index.php?m=plugins&c=Pushall&a=$1&id=$2 last;
rewrite ^(.*)$ /index.php?s=$1 last; break;
}
}

{/if}

建议填写1000，百度专用Sitemap自动乘以4倍。例：填写1000，百度sitemap就是4000条每页！

填写推送Token即可。结尾标红得那部分: http://data.zz.baidu.com/urls?site=https://www.xxxx.com&token=xxxxxxxx

百度多域名推送教程：推送额度较少的域名可以通过m.xxxx.com进行访问后台推送m手机端页面地址。这样就可以多推送m二级域名的额度！

填写神马站长平台的推送接口。如: https://data.zhanzhang.sm.cn/push?site=www.xxxx.com&user_name=xxxxxx@qq.com&resource_name=mip_add&token=TI_xxxxx

填写搜狗站长平台抓取的Cookie。示例：PHPSESSID=qbja3h8qfs6563hkuelivs8im2

填写搜狗站长提交链接页面的验证码。PS：务必填写最新获取的验证码！
